# SuperStore Sales Analysis - Data Science Project
This repository contains a complete data science case study on SuperStore sales prediction using Python.
End-to-end data science workflow analyzing SuperStore sales data with Python. Includes data cleaning, visualization, Linear Regression baseline, Random Forest improvement, cross-validation, and model comparison with feature importance.
## 📌 Project Overview
This project demonstrates an end-to-end data science workflow analyzing SuperStore sales data using Python.  
It covers data cleaning, visualization, baseline modeling with Linear Regression, and improvement with Random Forest.  
Cross-validation and error distribution plots are included to evaluate model performance.

## ⚙️ Workflow
1. Data loading & cleaning  
2. Exploratory Data Analysis (EDA)  
3. Visualization (sales trends, category breakdowns)  
4. Baseline model: Linear Regression  
5. Improved model: Random Forest  
6. Cross-validation  
7. Model comparison & visualization

## 📊 Results
- Linear Regression R² ≈ 0.61  
- Random Forest R² ≈ 0.81  
- Random Forest reduced MAE and RMSE significantly  
- Feature importance shows **shipping cost** as the strongest driver of sales predictions
